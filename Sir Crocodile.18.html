Sir Crocodile

เซอร์คร็อกโคไดล์

เซอร์คร็อกโคไดล์ ( Crocodile ) : คร็อโคไดล์ถือครอง ผลซูนะ ซูนะ ( Suna Suna ) ผลทราย สายโรเกีย ที่สามรถควบคุมทรายเเละเปลี่ยนร่างเป็นทรายได้ ปรากฎตัวครั้งแรกในอนิเมะตอนที่ 92 ( ภาคอลาบัสต้า ) คร็อกโคไดล์เป็นหัวหน้าขององค์กรอาชญากรรมลับที่ชื่อว่า บาร็อกเวิร์คส์ โดยจุดประสงค์ขององค์กรนี้คือการ ทำลายอาณาจักรแห่งทะเลทราย อลาบัสต้า โดยที่คร็อกโคไดล์ใช้นามเเฝงตนเองว่า Mr.0

คร็อโคไดล์มีบทบาทมากในภาค อลาบัสต้าเเละมีบทอีกทีคือ ภาคอิมเพลดาวน์เเละมารีนฟอร์ดตามลำดับ คร็อกโคไดล์นับว่าเป็นหนึ่งในเจ็ดเทพที่เเข็งเเกร่งมาก โดยที่สามารถทำให้ลูฟี่เกือบตายได้ถึงสองครั้ง ปัจจุบัน คร็อกโคไดล์ถูกปลดในฐานะเจ็ดเทพโจรสลัดเรียบร้อยเเล้ว

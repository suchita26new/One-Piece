Donquixote Doflamingo

ดองกี้โฮเต้ โดฟลามิงโก้

ดองกี้โฮเต้ โดฟลามิงโก้ ( Donquixote Doflamingo ) หรือ โจ๊กเกอร์ ( Joker ) : ผู้กินผลพารามีเซียอย่าง ผลอิโตะ อิโตะ (Ito Ito) ผลเส้นด้าย ปรากฏตัวครั้งเเรกนิอนิเมะตอนที่ 410

โดยมิงโก้มีบทบาทที่เป็นบาสใหญ่ของภาค เดรสโรสซ่า ( Dressrosa ) ที่ลูฟี่เอาชนะ โดฟลามิงโก้ได้ด้วย เกียร์ 4 โดฟลามิงโก้เป็นอีกตัวละครหนึ่งที่มี ฮาคิเเห่งราชันย์อีกด้วย หลังจากโดฟลามิงโก้เเพ้ลูฟี่ก็ถูกปลดจากตำเเหน่งเเละถูกส่งเข้าคุกนรกอิมเพลดาวน์

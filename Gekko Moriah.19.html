Gekko Moriah

เก็กโค โมเรีย

เก็กโค โมเรีย ( Gekko Moriah ) : ผู้กินผล คาเงะ คาเงะ ( Kage Kage ) ผลควมคุมเงา สายพารามีเซีย ที่สามรถดึงเอาเงาของคนอื่นไปใส่ในศพซึ่งจะกลายเป็นซอมบี้ได้หรือ นำมาใส่ร่างของตนเพื่อเพิ่มพลังได้ โมเรีย ปรากฏตัวครั้งเเรกใน อนิเมะตอนที่ 348 ที่ทริลเลอร์ บาร์ค ( Thriller Bark ) ที่เป็นเรือขนาดใหญ่ของตัวโมเรียเอง

โดยที่โมเรียมีบทบาทมากใน ภาคทริลเลอร์ บาร์ค เเละต่อมาที่โมเรียเข้าร่วมสงครามมารีดฟอร์ดโดยที่หลังจบสงครามโลก โมเรียโดนสั่งเก็บโดยรัฐบาลโลกเหตุเพราะพ่ายเเพ้ให้กับลูฟี่ ปัจจุบันโดนปลดจากตำเเหน่งเป็นที่เรียบร้อย
